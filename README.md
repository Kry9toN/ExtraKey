# Extra Key

# Usage
```bash
$ pkg install python
```
```bash
$ pkg install git
```
```bash
$ git clone https://github.com/Kry9toN/extrakey
```
```bash
$ cd Extrakey
```
```bash
$ python extrakey.py
```

```bash
$ wget https://raw.githubusercontent.com/Kry9toN/ExtraKey/master/extrakey.sh
```

```bash
$ ./extrakey.sh
```

```bash
$ sh -c "$(wget -O- https://raw.githubusercontent.com/Kry9toN/ExtraKey/master/extrakey.sh)"
```
